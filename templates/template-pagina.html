Ação principal
Estrutura base

Template para páginas internas do portal.

Use esta estrutura para páginas como cadastro de entidades, cadastro de doadores, listagem de itens, painel administrativo ou detalhes de estoque, mantendo a mesma identidade visual.

Bloco principal

Área de conteúdo da página

Substitua este trecho pelo conteúdo principal da página. Aqui pode entrar uma tabela, formulário, dashboard simples, listagem de entidades ou qualquer módulo funcional do sistema.

Campo Uso recomendado
Título Nome da página ou módulo
Subtítulo Contexto rápido para o usuário
Conteúdo Cards, tabelas, formulários ou gráficos simples
Bloco lateral

Resumo ou filtros

Este espaço funciona bem para filtros, indicadores resumidos, avisos operacionais ou ajuda contextual.

  • Resumo de cadastros
  • Status de atualização
  • Atalhos rápidos
  • Instruções de preenchimento
